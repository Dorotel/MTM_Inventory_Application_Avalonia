Future Updates

Roadmap items summarized from Planned_Implementations.md:

  • Centralized Exception Handling completion (normalization, keyboard shortcuts, Copy Details, optional Help).
  • Service interfaces and adapters: IVisualConnectionService, IAuthenticationService (prod), IInventoryService, IShopFloorService, IPartService, ILocationService, ISettingsStore, IAppDb.
  • Settings persistence via MAMP DB and “Test App DB” connectivity check.
  • MAMP schema/migrations for settings, audits, exception logs.
  • Dialogs enablement (Location Picker, Incomplete Part) and navigation polish.
  • Inventory Transfer implementation via VISUAL Toolkit in service layer.
  • Work Order Issue/Receipt implementation via service layer with TRACE rules.
  • Production authentication via VISUAL (OpenLocal/OpenLocalSSO).

See the phased delivery plan and acceptance checklist in the roadmap for details.
