MTM Inventory Application - Technical Documentation

Comprehensive technical documentation for the Avalonia .NET 8 desktop inventory management application with Infor VISUAL ERP integration.

Active Development Last updated: December 22, 2024 Version: 2.0
Documentation Modernization Complete! This documentation has been reorganized and modernized for improved navigation and accessibility. Plain English versions are now available for non-technical stakeholders.

Application Overview

MTM Inventory Application

Platform: Avalonia 11 on .NET 8 Desktop

Architecture: MVVM with Service Layer and Dependency Injection

Integration: Infor VISUAL ERP System

Features: Login Authentication, Inventory Transfer, Work Order Transactions, Exception Handling, Settings Management, App-owned Database (MAMP)

Quick Start for Developers

1. Environment Setup

  • .NET SDK 8.0+
  • Avalonia Templates: dotnet new install Avalonia.Templates
  • Visual Studio 2022 or VS Code with C# extension

2. Build and Run

dotnet restore
dotnet build --configuration Release
dotnet run --project MTM_Inventory_Application_Avalonia.Desktop

Quality Assurance

Critical Development Rules

  • Visual License Lifecycle: Always explicitly close/release VISUAL licenses after each operation
  • Error Handling: All UI-invoked methods must use centralized IExceptionHandler
  • Documentation Sync: Update CodeMap_CS_Files.md when modifying any .cs file
  • Interface Pattern: Depend on interfaces, not concrete implementations for services
