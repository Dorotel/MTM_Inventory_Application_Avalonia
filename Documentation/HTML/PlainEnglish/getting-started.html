Getting Started with MTM Inventory Application

Your step-by-step guide to logging in and using the inventory management system for the first time.

üìñ 10 minute read üìÖ Updated: December 22, 2024 Beginner friendly

Before You Begin

Make sure you have been given login credentials by your supervisor or IT department. You'll need your username and password to access the system.

  • Starting the Application
  • Logging In
  • Understanding the Main Screen

Step 1: Starting the Application

  • Find the Application Icon

    Look for the MTM Inventory Application icon on your desktop or in your programs menu. It should look like a purple square with "MTM" text.

  • Double-Click to Open

    Double-click the icon to start the application. You might see a loading screen for a few seconds - this is normal.

  • Wait for Login Screen

    The application will open and show you a login screen. This is where you'll enter your credentials to access the system.

    • üí°

    What if the icon isn't there?

    If you don't see the application icon, contact your IT department. They may need to install the software on your computer or give you access to it.

    Step 2: Logging In

    Login Information You'll Need

    • Username: This is usually your employee ID or a username assigned by IT
    • Password: Your secure password (keep this private!)
    • Site/Location: Your facility location (your supervisor will tell you which one to use)

  • Enter Your Username

    Type your username in the first box. Make sure there are no extra spaces before or after.

  • Enter Your Password

    Type your password in the second box. The letters will appear as dots (‚Ä¢) for security - this is normal.

  • Select Your Site (if needed)

    If you see a dropdown menu for "Site" or "Location," choose the one your supervisor told you to use.

  • Click "Login" or Press Enter

    Click the Login button or press the Enter key on your keyboard to sign in.

    • Login Problems? If you get an error message, double-check that you typed everything correctly. If it still doesn't work, contact your supervisor or IT support.

    Step 3: Understanding the Main Screen

    Once you successfully log in, you'll see the main application screen. Here's what each part does:

    üè†

    Main Menu Area

    This is usually on the left side or top of the screen. It shows the different tasks you can do, like "Inventory Transfer" or "Work Orders."

    üìä

    Work Area

    This is the large area in the middle where you'll do your actual work - entering information, viewing lists, and completing tasks.

    ‚ÑπÔ∏è

    Status Area

    Usually at the bottom of the screen, this shows helpful information like whether your last action was successful or if there are any messages.

    Common Buttons You'll See

    • "Save" or "Submit": Saves your work and applies changes
    • "Cancel": Cancels what you're doing without saving
    • "Search" or "Find": Looks up information in the system
    • "Clear": Erases what you've typed to start over
    • "Help": Shows help information for the current screen

    Step 4: Your First Task

    Let's try a simple task to get familiar with the system:

  • Look Around the Interface

    Take a minute to see what options are available in the main menu. Don't worry about clicking anything yet - just get familiar with what's there.

  • Try a Search (Safe to Practice)

    If you see a search box, try typing in a part number or item you know exists. This won't change anything - you're just looking.

  • Check the Help Feature

    Look for a "Help" button or menu. This will show you information specific to whatever screen you're on.

  • Practice Navigation

    Try clicking on different menu items to see what screens are available. You can always click "Back" or "Cancel" to return to where you started.

    • Don't Worry About Making Mistakes

    The system has built-in safeguards to prevent accidental changes. Most actions require you to confirm before anything is saved, so you can explore safely.

    Emergency Information

    • Stuck? Press the Escape key or look for a "Cancel" button

    • Can't find something? Use the search function or check the main menu
    • Application frozen? Wait 30 seconds, then contact IT if it doesn't respond
    • Need help now? Contact your supervisor or call IT support
