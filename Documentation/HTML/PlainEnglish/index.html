What is the MTM Inventory Application?

üì¶

Inventory Management Made Simple

The MTM Inventory Application is a desktop computer program that helps you manage inventory (parts, materials, and supplies) and process work orders in your manufacturing environment. Think of it as a digital assistant that connects to your main business system (Infor VISUAL) to help track and move inventory items.

Frequently Asked Questions

Do I need special training to use this application?

The application is designed to be user-friendly and intuitive. Basic computer skills and familiarity with your inventory processes are helpful. This documentation provides step-by-step guidance for all features.

What happens if I make a mistake?

The application has built-in safeguards to prevent common errors. If you do encounter an issue, most actions can be corrected through your normal business processes, and the system provides clear error messages to guide you.

How does this connect to our main system?

The application connects directly to your Infor VISUAL ERP system using secure, authorized connections. All data is synchronized in real-time, so changes appear immediately in both systems.

Can I use this on any computer?

The application is designed for Windows desktop computers and requires specific software components. Your IT team will handle installation and setup on authorized workstations.
